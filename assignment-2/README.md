# Assignment 2: API

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/license/mit/)

## Installation

install [node.js](https://nodejs.org/en/download/)

install [pnpm](https://pnpm.io/installation)

install dependencies

```sh
pnpm install
```

## Usage

build

```sh
pnpm build
```

run `dist/index.js`

```sh
pnpm start
```
or
```sh
node dist/index.js
```
